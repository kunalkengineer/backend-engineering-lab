# Branching Strategy

## Permanent Branches

```text
main
develop
```

## Feature Branches

```text
feature/module-name
```

Examples:

```text
feature/spring-core
feature/kafka-consumers
feature/redis-caching
feature/distributed-transactions
```

---

# Development Workflow

```text
feature/*
    ↓
develop
    ↓
main
```

---

# Commit Convention

Examples:

```text
feat(kafka): implement consumer groups
feat(redis): add cache aside strategy
feat(security): add jwt authentication
refactor(core): introduce service layer
docs(system-design): webhook retry architecture
```
